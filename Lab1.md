# Smart Log Analyzer

Проект по разработке ИИ-системы для автоматического анализа и сокращения логов ошибок (stack trace) с помощью локальной LLM (Ollama + llama3.2).

---

## Тема проекта

Разработка консольного приложения на Python, которое принимает на вход текстовый лог ошибки, анализирует его через локальную модель Ollama и выдаёт структурированный результат: тип ошибки, место возникновения, краткую суть и предполагаемую причину.

---

## План работ

| Лаба | Содержание |
|------|------------|
| 1 | Выбор темы, составление плана, создание git-репозитория |
| 2 | Установка Ollama, скачивание модели, написание базового скрипта с заглушкой |
| 3 | Интеграция с Ollama, разработка промпта, получение реального анализа |
| 4 | Улучшение вывода, обработка ошибок, форматирование результата |
| 5 | Добавление аргументов командной строки (чтение из файла) |
| 6 | Оформление README, инструкция по установке и запуску, запись видео-презентации |

---

## Стек технологий

- Python 3.9+
- Ollama (локальный сервер LLM)
- Модель: `llama3.2:3b` или `codellama:7b`
- Библиотека: `requests`
