# Smart Log Analyzer

Проект по разработке ИИ-системы для автоматического анализа и сокращения логов ошибок (stack trace) с помощью LLM.

---

## Тема проекта

Разработка консольного приложения на Python, которое принимает на вход текстовый лог ошибки, анализирует его через GigaChat API и выдаёт структурированный результат: тип ошибки, место возникновения, краткую суть и предполагаемую причину.

---

## План работ

| Лаба | Содержание |
|------|------------|
| 1 | Выбор темы, составление плана, создание git-репозитория |
| 2 | Настройка окружения, получение API-ключа GigaChat, написание базового скрипта с заглушкой |
| 3 | Интеграция с YandexGPT, разработка и тестирование промпта |
| 4 | Парсинг ответа от LLM, вывод структурированного результата в консоль |
| 5 | Обработка ошибок (невалидный лог, сбой API), добавление возможности читать лог из файла и аргументов командной строки |
| 6 | Оформление README (инструкция по установке и запуску), запись видео-презентации с демонстрацией работы в терминале |

## Как будет работать программа

```bash
# Запуск с вводом лога вручную
python main.py

# Или с чтением из файла (лаба 5)
python main.py --file error.log

# Или с прямой передачей лога (лаба 5)
python main.py --log "KeyError: 'user_id' at app.py line 42"
