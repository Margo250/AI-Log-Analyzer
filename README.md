# AI-Log-Analyzer

## Smart Log Analyzer

Система для автоматического анализа логов ошибок с помощью LLM.  
На вход подаётся stack trace, на выходе — структурированный разбор: тип ошибки, место, суть, причина.

## Как работает

1. Пользователь вставляет лог ошибки в текстовое поле
2. Система отправляет запрос к LLM API (YandexGPT / GigaChat / OpenAI)
3. Полученный ответ парсится и выводится в виде карточки
